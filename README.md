# Systemd-rc
конвектор команд для Open-RC.
конвектор не эмулирует systemd, а только переводит команды для systemd в openrc 

## установка
```
git clone https://b-e-n-z1342/Systemd-rc.git
cd Systemd-rc
./install
```

## Для QuasarLinux 
установка очень простая одной командой 

```
pkg -I systemd-rc
```
весь код на bash
